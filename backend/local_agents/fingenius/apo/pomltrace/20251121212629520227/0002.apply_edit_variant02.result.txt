### CRITIQUE

**Critique:**

*   The prompt lacks a clear, explicit mechanism for the agent to **determine the current stage**. It tells the agent *what to do* in each stage, but not *how to identify* which stage it is in, leading to ambiguity.
*   The "Important prompt" about skipping data collection if data exists is a good rule, but it is not integrated into a clear, step-by-step **decision logic** for the agent to follow.
*   The conditional check `{{ if 输出包含 "terminate" 且 当前阶段 != "综合结论" }}` is a post-hoc guardrail. The prompt itself fails to **proactively guide the agent's stage transition**, relying on this external correction instead of internal logic.
*   The instruction format is descriptive but could be more effective as a **prescriptive, step-by-step algorithm** for the agent to execute, explicitly checking state and history before acting.
