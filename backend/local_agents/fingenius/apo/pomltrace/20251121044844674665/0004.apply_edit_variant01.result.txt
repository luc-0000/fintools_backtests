# Critique

**Critique:**

*   **Missing Explicit State Tracking:** The prompt relies on the LLM to infer the "current stage" from the context, but it doesn't provide a clear, structured way to track or announce state transitions. This leads to ambiguity and incorrect tool usage.
*   **Ambiguous "Current State" Input:** The rule "根据当前状态" (based on current state) is too vague. The prompt fails to specify what information the "state" contains, leaving the LLM to guess whether data is already present.
*   **Weak Enforcement of Stage Progression:** The "important prompt" to skip data collection if data exists is a suggestion, not a hard rule. The prompt lacks a clear, conditional command structure (e.g., "IF data exists, THEN proceed to analysis").
*   **Inconsistent Formatting Enforcement:** The output format instructions (e.g., `[数据收集结果]`) are given as examples rather than strict requirements. This allows the LLM to deviate and produce unstructured responses that are harder to parse and validate.
