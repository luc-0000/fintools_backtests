### CRITIQUE

**Critique:**

*   **Missing Explicit State Tracking:** The prompt relies on the LLM to infer "当前状态" (current state) from the context, but it does not provide a clear, structured way to track which stage the process is in. This leads to ambiguity and incorrect tool usage.

*   **Insufficient Guardrails for Tool Availability:** The conditional check `{{ if 输出包含 "terminate" 且 当前阶段 != "综合结论" }}` is a post-hoc correction. The prompt lacks a proactive, explicit instruction at the *start* of each phase that clearly lists the allowed and forbidden actions.
*   **Ambiguous "Analysis" Output Format:** The instruction for the depth analysis phase, "仅允许生成分析文本," is too broad. It doesn't specify a required format (e.g., a report with headings, key findings) or content scope, which can result in low-quality or off-topic outputs that score poorly.
